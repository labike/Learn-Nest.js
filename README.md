
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## nest命令
- nest g co 生成controller
- nest g co --no-spec 不生成描述文件
- nest g s 生成service
- nest g module <module name> 生成module名称
- nest g class coffe/dto/create-coffe.dto --no-spec 生成对，例如coffeDto
